1 How this started

Ming sat down to pick up where he'd left off and asked one simple thing: "where's the link to the website?"

The honest answer was the spark for everything that followed:

There was no link yet. Your shelf existed as a design and a Google-Sheets prototype, but it had never actually been put online — nothing you could tap on your phone. So we decided to fix that, and make it nicer while we were at it.

At that point the app was: your 201 items across 51 brands (lots of blush 💗), a "Seoul Glass" Korean-cute design, and a back-end wired to a Google Sheet.

3 Talking to your own Claude Code

You don't need any of this just to use the app — open the link, sign in, done. But if you ever want to change it yourself with Claude Code, here's everything to hand it.

① Paste this into Claude Code to give it the full picture

I'm working on "Shelf Tracker" — my personal makeup-inventory app. It's already built and live; help me change or extend it. • Live app: https://shelf-tracker-kristina.web.app • Repo: https://github.com/zmzhong1/shelf-tracker (the whole app is web/index.html) • Firebase: project "shelf-tracker-kristina" — Firestore = database, Hosting = the website, Google sign-in = the lock Before editing, know this: - web/index.html is ONE self-contained file: plain HTML/CSS/JS + the Firebase "compat" SDK from gstatic. No build step, no React. - Data lives in Firestore collections `items` (the shelf) and `haul` (Korea shopping list). Each item has the usual fields plus `favorite` and `notes`. - The design is "Seoul Glass": pearl background, rosy + iridescent accents, Pretendard font, light by default with a dark toggle. - History is in docs/decisions.md; the redesign spec is in docs/superpowers/specs/. - To publish a change: npx firebase deploy --only hosting (add --only firestore if you touch firestore.rules) What I want to do: [describe your change — e.g. "add a 'finished' button to items", "make the favorite star gold", "show items expiring this month at the very top"].

② Access you'll need from Ming (ask him once)

  • Sign-in: your Google email is already on the app's allow-list — if it ever says "not on the allow-list," send Ming the exact Gmail you're using.
  • To edit the code: ask Ming to add you as a collaborator on the GitHub repo (or to make it public).
  • To publish changes yourself: ask Ming to add you to the Firebase project (Firebase Console → Project settings → Users and permissions).

③ One-time setup on your computer

  1. Install Claude Code, Node.js, and Git.
  2. Get the code: git clone https://github.com/zmzhong1/shelf-tracker.git
  3. Go into it: cd shelf-tracker
  4. Connect Firebase: npx firebase login (sign in with the Google account Ming added)
  5. Open that folder in Claude Code and paste the prompt from ① — then describe what you want.

On Windows the commands are the same — just run them in PowerShell.
